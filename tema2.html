TEMA 2

Grundlæggende Web

I dette tema fik vi kendskab til de grundlæggende værktøjer, som vi skal bruge som multimediedesignere.
Vi fik introduktion til programmeringssprogene HTML og CSS som vi skulle kode i programmet Visual Studio Code. Vi lærte bl.a. om grid, classes, margin, padding, responsivt design og media queries med tilgangen “mobile first”.
Vores proces dokumenterede vi i designprogrammet Figma og blev derudover undervist i ophavsrettigheder, gestaltlove, skrifttyper, styletiles, wireframes, tekstopsætning og billedformater.
Vi fik kendskab til redskaber som HTML- og CSS-validering som vi afslutningsvis benyttede til studiestartsprøven samt uploadede websitet gennem FileZilla.

PROJEKT

STUDIESTARTSPRØVE

Det afsluttende projekt i temaet gik ud på at kode et responsivt website som var vores studiestartsprøve. Vi fik udleveret layoutdiagrammer og wireframes som vi skulle følge samt benytte bestemte billeder og tekst, men besluttede selv fonte og farver.
